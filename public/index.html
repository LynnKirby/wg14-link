wg14.link

An unofficial ISO C (WG14) redirect service.

Usage

wg14.link/nXXXX
Get document.
wg14.link/{c99,c11,c18,c2x}
Get latest working draft.

Experimental

Work-in-progress features.
The routes work but the data is inaccurate.

wg14.link/nXXXX.bib
Get BibTeX citation.
wg14.link/nXXXX.yaml
Get CSL-YAML citation.
wg14.link/index.bib
wg14.link/index.yaml
Get everything.
